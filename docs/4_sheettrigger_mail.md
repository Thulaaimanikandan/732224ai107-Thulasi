# Steps

- Sign in to your automation platform.
- Click "Create Workflow".
- Add "Google Sheets Trigger" and select **Row Added**.
- Add "Send a message"(Gmail).
- Map the sheet data to the email fields.
- Execute each step to test.
- Run the complete workflow.
