# Steps
- Log in to n8n.
- Create a workflow.
- Add the first step: Search for “Schedule Trigger” and adjust your preferences.
- Execute the step.
- Search for “Email” and select “Send Email Message.”
- Execute the step.
- Finally, execute the entire workflow.
