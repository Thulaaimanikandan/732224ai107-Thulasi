# Steps
- Sign in to the platform.
- Click Create Workflow.
- Add On form submission as the first step.
- Add AI Agent.
- Select Google Gemini Chat Model as the Chat Model.
- Add Send a message (Gmail).
- Configure the email details.
- Execute each step.
- Run the full workflow.
