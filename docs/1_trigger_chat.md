# Steps
- Create workflow
- Add first step
- Search "Chat Trigger" and select it
- Click "+" button to add another node
- Search "Gmail"
- Select "Send a message"
- Enter your Gmai account
- Enter the receiver Gmail address
- Enter the subject and message you want to send
- Click Execute Workflow
